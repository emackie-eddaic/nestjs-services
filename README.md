# nestjs-services

Monorepo for NestJS modules packages (published to [npm](https://www.npmjs.com)). Currently supporting:

- [@eddaic/nestjs-cacheable](https://github.com/emackie-eddaic/nestjs-services/tree/main/packages/nestjs-cacheable)
- [@eddaic/nestjs-google-maps](https://github.com/emackie-eddaic/nestjs-services/tree/main/packages/nestjs-google-maps)
- [@eddaic/nestjs-maxmind-geoip2](https://github.com/emackie-eddaic/nestjs-services/tree/main/packages/nestjs-maxmind-geoip2)
- [@eddaic/nestjs-thingsboard-http-integration](https://github.com/emackie-eddaic/nestjs-services/tree/main/packages/nestjs-thingsboard-http-integration)
